# Список КП

Список всех КП пространства. Точка входа для работы с коммерческими предложениями — отсюда создаются новые КП и открываются существующие.

## Столбцы списка

| Столбец | Описание |
|---|---|
| Компания | Название компании клиента |
| Комната | Название КП |
| Владелец | Инициалы менеджера-владельца |
| Тренд | Статус КП: Черновик, Опубликован, Архив |
| Источник | Всегда «Шаблон» |
| Последний просмотр | Дата последнего открытия страницы клиентом |
| Просмотры | Количество открытий клиентом |
| Действия | Количество действий клиента на странице |
| Время | Суммарное время просмотра клиентом |
| Изменено | Дата последнего изменения КП |

## Статусы КП

| Статус | Описание | Что можно делать |
|---|---|---|
| Черновик | КП создано, но не опубликовано. Клиент не имеет доступа | Редактировать, опубликовать, удалить |
| Опубликован | КП доступно по ссылке. Аналитика активна | Редактировать, снять с публикации |
| Архив | КП снято с публикации. Клиент больше не имеет доступа | Редактировать, опубликовать снова, удалить |

:::caution
Удалить можно только КП в статусе **Черновик** или **Архив**. Опубликованное КП удалить нельзя — сначала нужно перевести его в архив.
:::

## Сортировка

Список сортируется автоматически:
1. Опубликованные — сверху, по дате изменения (новые первые)
2. Черновики — в середине
3. Архив — снизу

Ручная сортировка и фильтрация недоступны.

## Видимость

- **Менеджер** — видит только свои КП
- **Администратор и Владелец** — видят КП всей команды

## Создание КП

Кнопка «Создать КП» запускает двухшаговый мастер: выбор шаблона → заполнение анкеты.
