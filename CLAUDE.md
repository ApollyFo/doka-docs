# Project: Doka

Контекст продукта **ДОКА** (коммерческие предложения, шаблоны КП, роли, аналитика). Этот репозиторий — **пользовательская документация** (сценарии и логика), а не исходный код приложения.

## Stack

- **Этот репозиторий (`doka-docs`)**: статический сайт на [Docusaurus](https://docusaurus.io/) 3, React 19, TypeScript, Node.js ≥ 20.
- **Backend**: в этом репозитории не описан; смотри основной репозиторий приложения Doka.
- **DB**: в этом репозитории не описана; смотри основной репозиторий приложения Doka.

## Architecture

- **Agent pipeline** (контекст автоматизации/агентов продукта): **Interviewer → Researcher → Validator**.

## Build & Run

- Установка: `npm install`
- Локальная разработка: `npm run dev` (алиас Docusaurus `start`; эквивалентно `npm start`)
- Сборка: `npm run build`
- Продакшен-просмотр билда: `npm run serve`

## Key conventions

- **Документация**: Markdown и front matter в `docs/`; навигация — `sidebars.ts`.
- **Основной код Doka** (вне этого репозитория): JSON-схемы — в `/schemas`; агенты — в `/agents` (пути относительно корня репозитория приложения).

## Язык и локаль

- Локаль сайта по умолчанию: `ru` (`docusaurus.config.ts`).
