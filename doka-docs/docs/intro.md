# ДОКА — Документация продукта

Документация для команды разработки. Описывает пользовательские сценарии и логику работы системы.

## Разделы

- **Шаблоны** — создание и редактирование шаблонов КП
- **КП** — создание коммерческих предложений из шаблонов
- **Аналитика** — отслеживание активности клиентов
- **Роли и права доступа** — матрица прав
